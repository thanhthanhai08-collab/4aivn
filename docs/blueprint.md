# **App Name**: Clean AI Hub

## Core Features:

- AI Tool Ranking: Display a curated ranking of top AI tools based on user votes and popularity.
- AI News Feed: Present the latest news and articles related to AI advancements.
- Demo Chatbot: A demo chatbot interface with rule-based logic.
- User Authentication: User authentication via Google or email to enable tool rating and favorite-saving features.
- Filtering and Search: Enable users to filter AI tools based on categories.
- Responsive Design: Responsive design that adapts to different screen sizes, ensuring a consistent experience across devices.

## Style Guidelines:

- Primary color: Dark blue (#0B69FF) for calls to action, creating a sense of trust.
- Background color: Light gray (#F5F5F5) for a clean and minimalist aesthetic.
- Accent color: Light purple (#E8E6FB) for highlights, adding a modern touch.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and readable user experience.
- Simple and modern icons to represent different AI tool categories.
- A clean, grid-based layout similar to Airtable, providing a user-friendly way to explore AI tools and news.
- Subtle transitions and animations for a smooth and engaging user experience.
